About Portfolio

Nikah Plus – Marriage Matching App

Nikah Plus is a modern and user-friendly mobile application designed to help individuals find suitable marriage partners based on compatibility, preferences, and values. The app offers features like profile creation, interest selection, secure authentication, and match discovery through a clean and intuitive UI.

I worked on building the UI using Flutter, implemented secure user registration and login with Firebase Authentication, and handled real-time data using Cloud Firestore. Special focus was given to smooth user experience, privacy, and mobile responsiveness.

Sparkle – Modern Dating App

Sparkle is a sleek, interactive dating app built using Flutter, designed to connect like-minded individuals through intelligent matching, engaging UI, and smooth user experience. The app includes features like gender-based onboarding, interests & personality selection, voice/video introduction, and profile discovery.

I contributed to designing and developing the front-end using Flutter, implemented smooth screen transitions, and handled user input forms with proper validations. Firebase was used for user authentication and real-time data storage.
